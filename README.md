# 🧮 SmartCalc – All-in-One Calculator

SmartCalc is a Streamlit web application that combines basic mathematical and financial calculations into a single interactive tool.


## Live Application

🔗 **Deployment Link**

https://smart-calculator-z2nwr2rfta3fzb5b3uctz4.streamlit.app/


## Features
- Basic Arithmetic Operations
- Percentage Calculator
- Simple Interest Calculator
- Compound Interest Calculator
- EMI Calculator
- User-friendly interface

## Tech Stack
- Python
- Streamlit

## How to Run Locally
```bash
pip install streamlit
streamlit run app.py
